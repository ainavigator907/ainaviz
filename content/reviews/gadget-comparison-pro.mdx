## はじめに：「個人のレビュー」と「法人購入」は別物

Amazonや個人ブログのPC評価は「個人が使いやすいか」が基準です。
しかし法人で何十台・何百台と導入する場合、**評価軸が根本から変わります**。

総務部長として延べ数百台のPC選定に関わってきた経験から、
「稟議が通る＝導入後も問題が起きにくい」選択基準を共有します。

## 法人PC選定の5つの評価軸

### 1. TCO（総所有コスト）

購入価格だけでなく、3〜5年間の運用コストで比較します。

| 項目 | 含まれるコスト |
|------|-------------|
| 初期費用 | 本体・周辺機器・ライセンス |
| 運用費 | 保守契約・修理費・消耗品 |
| 管理費 | MDM・IT管理工数（人件費換算） |
| 廃棄費 | データ消去・リサイクル費用 |

### 2. 管理のしやすさ（IT部門視点）

社員が増えるほど、**台数×管理コスト**が効いてきます。

| OS | MDM対応 | 管理のしやすさ | コスト |
|----|---------|-------------|-------|
| Windows 11 Pro | Intune/JAMF | ◎ | 標準 |
| macOS | JAMF | ○（追加費用あり） | 割高 |
| ChromeOS | Google Admin | ◎ | 低コスト |

### 3. セキュリティ対応

情報漏洩事故1件のリスクを考えると、セキュリティ機能への投資は合理的です。

- **BitLocker（Windows）**：フルディスク暗号化が標準
- **TPM 2.0搭載**：ハードウェアレベルのセキュリティ
- **顔認証・指紋認証**：パスワード管理のコスト削減

### 4. サポート体制

**翌営業日訪問修理**は法人選定で必須に近い条件です。
1日業務が止まると、人件費換算でPCの価格を上回るケースがあります。

### 5. 拡張性・標準化

複数台を管理するなら、**同一機種・同一スペックでの統一**が運用効率を上げます。
「安い機種とハイスペック機種の混在」は、管理コストが増大します。

## 用途別おすすめ

### 一般事務職（Excel・Word・Teams中心）
**ThinkPad X1 Carbon i5モデル**

理由：安定した法人サポート・軽量・バッテリー持続・合理的な価格帯。
Lenovoの法人向けサポートは国内トップクラスで、修理対応が速い。

### エンジニア・デザイナー（高負荷作業）
**MacBook Pro M4 Pro**

理由：M4 Proの処理性能はWindows競合を圧倒。
映像編集・コーディング・機械学習なら断然コスパが良い。
ただし、Windowsオンリーのシステムが多い環境では注意が必要。

### 外出が多い営業職・役員
**VAIO Pro PG**

理由：689gという国内最軽量クラス。
1日中持ち運ぶ用途では「1kg以下」が生産性に直結する。

### タブレット兼用が必要な現場
**Surface Pro 11**

理由：タブレットモードでの直接入力が可能。
製造現場・医療・建設など、机のない場所での使用に最適。

## 稟議を通すためのポイント

1. **TCOで計算する**：「高い」と言われたら5年間のTCOを提示
2. **セキュリティリスクで語る**：「安い機種でデータ漏洩が起きた場合の損失」と比較
3. **管理工数を人件費換算**：「IT管理に毎月○時間 × 時給 = ○円」を見せる
4. **他社事例を引用**：同業他社の導入事例は説得力が高い

## まとめ

「個人で気に入ったPCを買う」と「組織で導入するPCを決める」は
全く異なる意思決定です。

TCO・管理コスト・サポート体制を軸に評価することで、
「現場が使いやすく、IT部門が管理しやすく、経営が承認しやすい」
三方良しの選択ができます。
